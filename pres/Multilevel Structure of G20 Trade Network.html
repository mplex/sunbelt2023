Multilevel structures

Use "multiplex" package for constructing multilevel structures, which in this case are in two flavours:

  • Actor co-affiliations
  • Binomial projection of actors and events.

Positional analysis multilevel

For the algebraic analysis, construct first a positional system of the G20 countries network with the clustering information bridge countries affiliations.

# vector, counter, and just non-overlapping affiliations in 'G20b'
club <- vector(length=nrow(G20b))
k <- 1L
for(i in seq(3,5)) {
    club[which(G20b[,i]==1)] <- k
    k <- k + 1L  
}
# Positional system for actors with clustering information
G20bn <- G20net[bridges, bridges, ] |> 
  multiplex::reduc(valued=TRUE, lbs=c("G7.C","BRICS.C","MITKA.C"), clu=club)
